# LLM Resources

These files are designed for AI coding assistants (Claude, ChatGPT, Cursor, Copilot, etc.). Paste any of these URLs into your AI tool to integrate dTelecom video conferencing into your app.

## Documentation for AI

| File | Description |
|---|---|
| [llms.txt](https://docs.dtelecom.org/llms.txt) | Compact index of all docs with one-line summaries — start here |
| [llms-full.txt](https://docs.dtelecom.org/llms-full.txt) | Complete documentation in a single file — all guides, APIs, code examples |
| [prompt.txt](https://docs.dtelecom.org/prompt.txt) | System prompt for AI agents — architecture, SDK patterns, code templates |
| [CONFERENCE_APP_PLAN.md](https://github.com/dTelecom/docs/blob/main/CONFERENCE_APP_PLAN.md) | Full blueprint for building a Google Meet-style app — architecture, file structure, every component |

## Example App — Created 100% by AI Agent

[**dTelecom Meet**](https://github.com/dTelecom/dtelecom-meet) is a complete video conferencing app created 100% by an AI agent using the documentation above.

**[Live Demo](https://dtelecom-meet-demo.vercel.app/)** | **[Source Code](https://github.com/dTelecom/dtelecom-meet)** | **[Deploy Your Own](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FdTelecom%2Fdtelecom-meet&env=API_KEY,API_SECRET&envDescription=Get%20your%20API%20keys%20from%20cloud.dtelecom.org&envLink=https%3A%2F%2Fcloud.dtelecom.org&project-name=dtelecom-meet&repository-name=dtelecom-meet)**

Features: video/audio conferencing, pre-join preview, host controls, in-room chat, live participant count via webhooks, one-click Vercel deploy.

## How to Use

**Claude / ChatGPT:** Paste `https://docs.dtelecom.org/llms.txt` into the chat.

**Cursor / Copilot:** Add `https://docs.dtelecom.org/llms-full.txt` to your project context.

**Custom AI agent:** Use `prompt.txt` as the system prompt for a dTelecom-aware coding assistant.
